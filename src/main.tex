\section{Fórmulas Matemáticas}
Una de las mayores ventajas de usar \LaTeX{} es la facilidad para escribir matemáticas con excelente calidad tipográfica. Aquí tienes un par de ejemplos clásicos:

La famosa ecuación de equivalencia entre masa y energía de Einstein:
\begin{equation}
    E = mc^2
\end{equation}

Y la fórmula general para resolver ecuaciones de segundo grado:
\begin{equation}
    x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
\end{equation}
